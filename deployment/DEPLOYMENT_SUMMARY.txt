RENDER.COM DEPLOYMENT (5 minutes):

1. Prepare your code:
   git add .
   git commit -m "Ready to deploy"
   git push origin main

2. Go to render.com
   - Sign up with GitHub
   - Click "New Web Service"
   - Select your repository
   - Fill in: Name, build command (npm install), start (npm start)
   - Choose Free plan
   - Deploy

3. Get your URL (e.g., https://peerconnect-signaling.onrender.com)

4. Update your code:
   In webapp/public/lib/signaling.js:
   const SIGNALING_SERVER = 'wss://your-app.onrender.com/signal';

5. Deploy frontend on Vercel:
   - Go to vercel.com
   - Import your GitHub repo
   - Auto-deploy

6. Test!
   - Open frontend URL
   - Create room
   - Share with friend
   - Should connect in seconds

DIGITAL OCEAN DEPLOYMENT (20 minutes):

1. Create DigitalOcean account
2. Create a Droplet ($5/month, Ubuntu 22.04)
3. SSH into it: ssh root@ip
4. Clone repo: git clone [your-repo]
5. Run: sudo ./deploy-self-hosted.sh yourdomain.com
6. Point domain DNS to droplet IP
7. Done! App runs at https://yourdomain.com
